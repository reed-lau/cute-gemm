# TMA Descriptor Test Dataset
# ============================
# Format: dtype swz interl l2 oob ndim  shape...  stride(bytes)...  box...  es...
#
# Total params per line: 6 + 4*ndim
# Stride is in BYTES and determines dimension ordering for TMA.
# All arrays (shape, stride, box, es) are in the same user-given order.
# The program internally sorts by stride ascending (smallest = innermost).
#
# dtype:   0=U8 1=U16 2=U32 3=I32 4=U64 5=I64 6=F16 7=F32 8=F64 9=BF16
# swz:     0=NONE 1=32B 2=64B 3=128B
# interl:  0=NONE 1=16B 2=32B
# l2:      0=NONE 1=64B 2=128B 3=256B
# oob:     0=ZERO 1=NAN
#
# Usage:
#   make test
#   cat tests.txt | grep -v '^#' | grep -v '^$' | xargs -L1 ./main
